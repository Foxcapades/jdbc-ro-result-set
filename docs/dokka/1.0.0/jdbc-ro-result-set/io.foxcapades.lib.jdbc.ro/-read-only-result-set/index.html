Read Only ResultSet

Thin, read only wrapper around a given result set that blocks actions that mutate the state of the underlying ResultSet instance.

All mutating methods will throw an UnsupportedOperationException

Author

Elizabeth Paige Harper - https://github.com/foxcapades
